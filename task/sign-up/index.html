The purpose of this task is to create a responsive sign-up page.

Problem Description

Implement the following design draft using HTML/CSS. The draft is a rough blueprint for a desktop version - The implementation doesn't need to be pixel perfect. Focus on cross-browser compatibility and follow mobile-first practises.

Instructions

  • Download background image.
  • Create a index.html and a main.css file. Include all necessary assets.
  • No form action, no form validation required.
  • Send solution as a zip file.

Requirements

  • Use semantic HTML5/CSS3* syntax.

    *If you're using a CSS preprocessor, such as Less or Sass, please include the source files as well.

  • Responsive Design (320px to 1200px).
  • Supports for all modern browsers (IE9+).
  • Spend max. 3h. Complete within a week (two weeks in special circumstances).

We hope you have fun working on this mini-project and are looking forward to examine your solution.
